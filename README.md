# 🎬 Y2P - Free YouTube Converter

## 🌟 About

**Y2P** is a fast and elegant web app for converting YouTube videos to **Code**. Designed with a sleek dark-red theme, glowing animations, and full responsiveness, it delivers a modern experience for users.

---

## 🚀 Features

- 🎨 Dark red modern UI with red glowing animations
- 🔁 Simulated conversion progress
- 🎯 Download-ready view with video preview
- 💻 Built using Angular + Bootstrap 5
- 📱 Responsive design for all devices

---

## 🛠️ Tech Stack

- ⚙️ Angular 14+ (standalone components)
- 🎨 Bootstrap 5
- 💡 HTML5, CSS3, TypeScript
- 📦 Node.js (for serving locally)
